# nuxt-web
#### A: 本项目基于nuxt搭建，主要是做PC + M端（移动端）适配 ，服务端渲染SSR，响应式官网开发
#### B: 如果你对本项目感兴趣或对你有帮助，欢迎Star！！！

### 一、项目运行
- > *npm install*  ( 可简写：*npm i* )
- > *npm run dev*
